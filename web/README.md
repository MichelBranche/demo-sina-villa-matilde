# Sina Villa Matilde — Demo React (Awwwards-style)

Esperienza immersiva per [Sina Villa Matilde](https://www.sinahotels.com/it/h/sina-villa-matilde-torino/): scroll fluido, tipografia editoriale, animazioni GSAP.

## Avvio

```bash
cd web
npm install
npm run dev
```

## Stack

- React 19 + Vite + TypeScript
- Tailwind CSS v4
- GSAP + ScrollTrigger + `@gsap/react`
- Lenis smooth scroll
- Cursor custom, preloader, horizontal scroll pin

## Note

- `prefers-reduced-motion` disattiva animazioni pesanti e cursor custom
- Media in `public/media/`
- Inventario contenuti in `../inventory/`
